# 🛠️ SolidWorks Automation: Release Package Export

[cite_start]**Author:** Tristan JACQ [cite: 4]
**Current Version:** v1.25

## 📝 Overview
* [cite_start]This SolidWorks macro is an automation tool developed to standardize and accelerate the creation of production release packages[cite: 9]. 
* [cite_start]It ensures consistency and traceability of exported data while automatically structuring the storage hierarchy on the server[cite: 11].

## ✨ Key Features
* [cite_start]**Multi-format Export:** Simultaneous generation of PDF, DXF, and STEP files bundled into a single ZIP archive[cite: 15].
* [cite_start]**Intelligent Naming:** Automatic file naming combining drawing number, revision index, and the current date[cite: 16].
* [cite_start]**Recursive Processing:** Automatic traversal of Bills of Materials (BOM) to generate deliverables for every assembly component[cite: 20].
* [cite_start]**Automatic Archiving:** History management by automatically moving older versions to a dedicated folder[cite: 23].

## 📂 Repository Content
* `main`: Contains the full macro history (from v1.0 to v1.25).
* [cite_start]`lightweight-version`: A simplified version for quick local exports without BOM processing[cite: 70].
